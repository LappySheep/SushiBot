SushiBot

A general-purpose bot developed by Sushi#4347.



Commands

s?help Displays help message
s?info Basic bot info
s?movie [movie/show] Displays info about the given movie or show
s?weather [location] Weather data for that location
s?mc [server ip] Displays info about the given server
s?bazaar Shows profitable bazaar flips
s?joke Displays a random dad joke
s?meme Displays a random meme
s?fact Displays random useless fact
s?hypixel [user] Hypixel data for the user
s?item [item] Price data for the item
s?cat Random cat photo
s?dog Random dog photo
s?say [text] Displays the text
